¿ Cómo mintear NFTs ?

Las siguientes instrucciones están diseñadas para ayudarlo en el proceso de creación de un NFT, o sea, mintear. Verá estos, basados ​​en los marketplaces de las Sisterchains de EOS:

Proton Marketplace​

Ir a Protonmarket.com para poder conectar nuestra billetera, en este caso hemos seleccionado WebAuth.

Next

AreaX NFT​

Ir a Areaxnft.com para poder conectar nuestra billetera, en este caso hemos seleccionado Anchor Wallet.

Next

AtomicHub​

Ir a wax.atomichub.io para poder conectar nuestra billetera, en este caso hemos seleccionado Anchor Wallet.
